# go-hello-world

This is an example repo.

## Usage Instructuctions

Install

```
go get github.com/bluecodemonks/go-hello-world
```

Run

```
$ go-hello-world
Hello World
```

Uninstall

```
$ rm $GOPATH/bin/go-hello-world
```
